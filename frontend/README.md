# ParkSpot | Sistem Pengelolaan Parkiran Mobil

Sistem antarmuka pengguna modern untuk mengelola ketersediaan dan pemesanan tempat parkir secara real-time.

## ✨ Fitur Utama

- **Denah Interaktif**: Visualisasi tempat parkir menggunakan Konva.js dengan indikator warna real-time.
- **Sistem Pemesanan**: Formulir pemesanan dengan validasi data.
- **Monitoring Real-time**: Countdown waktu parkir dan indikator overtime.
- **Data Persistent**: Menggunakan LocalStorage untuk menyimpan data pemesanan secara lokal di browser.
- **Desain Responsif**: Optimal untuk tampilan Desktop, Tablet, dan Smartphone.
- **Aesthetic Premium**: Desain bersih dengan tipografi modern dan efek glassmorphism.

## 🛠️ Stack Teknologi

- **Frontend**: React.js
- **Graphics**: Konva.js (React-Konva)
- **Styling**: Vanilla CSS (Custom Design System)
- **Icons**: Lucide React
- **Date Management**: Date-fns

## 🚀 Cara Menjalankan

1. Clone atau download folder proyek ini.
2. Buka terminal di direktori proyek.
3. Jalankan perintah: `npm install`
4. Jalankan perintah: `npm run dev`
5. Buka link yang muncul di terminal (biasanya `http://localhost:5173`).

## 📁 Struktur Proyek

- `src/components`: Komponen UI (Map, Form, Details).
- `src/hooks`: Custom hooks (useLocalStorage).
- `src/constants`: Data awal dan konfigurasi.
- `src/App.jsx`: Logika utama dan layout.
- `GUIDE.md`: Panduan implementasi dan referensi belajar.
